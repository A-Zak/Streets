Streets
=======

Streets of Tel-Aviv

Setup
========

install deps

```
npm install
```

start the server

```
node index.js
```

load stories into db

```
GET /loadStories
```

Supported APIs

get a random story

```
GET /story
```

get a story by ID

```
GET /story/story_id
```

create a story

```
POST /story
```
